## 📡 服务器连接信息

所有模式使用同一个 IP 地址进入，进入大厅后使用命令切换。

:::tip[一键复制]
`bangdream.lazyalienserver.top`
:::

## 🛠️ 选择你的游玩模式

<Tabs>

<TabItem label="生存 & 创造服 (推荐)" icon="star">

**适用版本：Minecraft 1.21.7 (Fabric)**

<Steps>

1. **安装游戏核心**

   请使用 PCL2 / HMCL 等启动器，下载安装 **Fabric 1.21.7** 版本。

2. **安装必要模组**

   为了获得最佳体验，请下载以下模组放入 `.minecraft/mods` 文件夹。

   | 模组名称 | 功能说明 | 下载链接 |
   | :--- | :--- | :--- |
   | **Xaero's Minimap** | 小地图 | [Modrinth 下载](https://modrinth.com/mod/xaeros-minimap) |
   | **Xaero's World Map** | 世界地图 | [Modrinth 下载](https://modrinth.com/mod/xaeros-world-map) |
   | **ViaFabricPlus** | 跨版本兼容 | [Modrinth 下载](https://modrinth.com/mod/viafabricplus) |
   | **AllMusic** | 点歌工具 | 📂 请加群下载 |

   :::note[关于版本]
   如果你使用的是更高版本的客户端（如 1.21.x），必须安装 **ViaFabricPlus** 才能进入服务器。
   :::

3. **切换服务器**

   进入游戏大厅后，在聊天栏输入以下命令切换：

   *   **生存服**：`/server survival`
   *   **创造服**：`/server creative`

</Steps>

</TabItem>

<TabItem label="特别行动 (枪战)" icon="sword">

**适用版本：Minecraft 1.16.5** (或使用高版本 + 兼容模组)

<Steps>

1. **准备环境**

   *   **方案 A (推荐)**：直接下载 1.16.5 Fabric 客户端。
   *   **方案 B**：使用 1.21 客户端 + ViaFabricPlus 模组。

2. **加载资源包**

   *   枪战模式需要专用的材质包才能看到枪械模型。
   *   📥 **获取方式**：请加入 QQ 群文件下载 `特别行动资源包.zip`。

3. **进入战场**

   在大厅输入以下命令：`/server minigames`

</Steps>

</TabItem>

</Tabs>
